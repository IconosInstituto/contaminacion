## La basura en los oceános

Proyecto Integrador de cuarto cuatrimestre.

---

El siguiente proyecto se enfoca en *concientizar* a la gente que **tira basura** en las playas.

La *materias* involucradas son las siguientes:

- Sonido Digital
- Css
- Diseño de espacios
- Dibujo 3D
- Significación
- Animación
